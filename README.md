# Modbus Logger

## Details
This Modbus (RS485) Logger is Intended to run on an ESP32 based MCU and collect data from Eastron 120M Power Meters. The Data then gets sent to an RPI Zero where it gets further processed/stored

## Hardware
- Norvi
- RPI Zero

## Requirements
- ESP-IDF
- ESP-Modbus
